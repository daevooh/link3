Understanding Tokens on Link3
What are tokens?

Tokens are digital assets that represent value, access rights, or ownership on a blockchain network. With Link3, you can create and manage tokens for various use cases including:

  • Utility tokens for your application's ecosystem
  • Reward tokens for incentivizing user behaviors
  • Access tokens for premium features
  • Governance tokens for community decision making
Token Standards

We currently support the following token standards:

Standard Description Networks
ERC-20 Standard fungible token for Ethereum and compatible networks Ethereum, Polygon, Binance Smart Chain, Arbitrum, Optimism
ERC-721 Non-fungible token (NFT) standard Ethereum, Polygon, Binance Smart Chain
ERC-1155 Multi-token standard supporting both fungible and non-fungible tokens Ethereum, Polygon
Token Creation Process
Step 1: Request Creation

Submit a token creation request with the following information:

  • Token name and symbol
  • Total supply
  • Blockchain network
  • Admin address (for token management)
  • Token features (optional)
Step 2: Review & Approval

Our team reviews your request to ensure:

  • Compliance with regulations
  • Adherence to network standards
  • Security best practices
  • Valid configuration parameters
Step 3: Deployment

Upon approval, we:

  • Deploy your token contract to the selected network
  • Initialize with your provided parameters
  • Perform security verification
  • Transfer ownership to your admin address
Step 4: Integration

After deployment:

  • Access your token via the contract address
  • Integrate with your application using our SDK
  • Monitor token activities via our dashboard
  • Access token functions through our API
Frequently Asked Questions

Token creation costs vary by network:

  • Testnet deployments: Free
  • Mainnet deployments: Network gas fees + service fee (see pricing page)

For enterprise customers, custom pricing plans are available.

Certain token parameters are immutable after deployment, including:

  • Token name and symbol
  • Decimals
  • Total supply (unless mintable feature is enabled)

If you require upgradeable tokens, select the "upgradeable" option during token creation.

  • Testnet deployments: Usually within 24 hours
  • Mainnet deployments: 2-5 business days (includes security review)

Premium customers receive expedited processing.
