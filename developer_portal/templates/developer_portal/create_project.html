About Projects

Projects are the way you organize your integration with Link3. Each project represents an application or service that will use tokenization.

What happens when you create a project:
  • A unique API key is generated for your project
  • Your project is set up with off-chain rewards
  • You can configure tokenization rules specific to this project
  • User interactions and rewards are tracked immediately
Tip: Use separate projects for development and production environments.
Next Steps

After creating your project, you'll want to:

  1. Configure your token settings
  2. Set up tokenization rules
  3. Integrate the API in your application
  4. Monitor analytics and user engagement

Our dashboard will guide you through each step of the process.
